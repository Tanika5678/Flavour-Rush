🍔 FlavourRush – Online Food Ordering Website

FlavourRush is a modern and user-friendly food ordering web application designed to provide an engaging browsing experience for users who want to
explore different cuisines and food items. The website allows users to view a variety of dishes, navigate through multiple food categories, and add 
selected items to a cart.
The main objective of this project is to demonstrate frontend web development skills, including responsive layout design, user interface structuring, 
and interactive features using JavaScript.
This project focuses on building a clean and organized structure while ensuring smooth navigation between pages such as the home page, menu page, and cart page.

✨ Key Features
Interactive and visually appealing user interface

Multiple food categories including burgers, pizza, pasta, sushi, drinks, and desserts

Menu page displaying food items with images and details

Add-to-cart functionality for selected items

Organized navigation between different pages

Clean and responsive layout for better user experience

🛠️ Technologies Used
This project is built using the following web technologies:
HTML5 – Structure of the website
CSS3 – Styling and layout design
JavaScript – Interactive functionality and cart behavior
Images & Media Assets – Visual representation of food items

* Project Structure: 
FlavourRush/
│
├── index.html        # Landing page of the website
├── home.html         # Main home interface
├── menu.html         # Displays food menu and categories
├── cart.html         # Shopping cart for selected items
│
├── style.css         # Styling and design of the website
├── app.js            # JavaScript logic for interactions
│
└── images/           # Contains all food item images

🚀 How to Run the Project
Follow these simple steps to run the project locally:
Clone the repository

git clone https://github.com/yourusername/FlavourRush.git
Open the project folder.

Run the application by opening index.html in any web browser.
No additional installations or dependencies are required.
